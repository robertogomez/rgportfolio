Responsive Web Design

Dec. 23, 2014

I've updated the site to follow modern responsive web design patterns. Built upon the Bootstrap framework, my portfolio is now easily accessible from mobile devices. Using the framework's grid system, the site's layout contracts when viewed from small-screened devices, such as smartphones and tablets, and expands for larger ones such as laptops and desktops.
